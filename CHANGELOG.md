# 更新日志

## [v1.4.0] - 2026-04-26

### 新增功能
- **多人模式** - 新增多人同屏成语接龙游戏模式，支持添加多名玩家并实时展示分数和专属头像
- **计分与排名** - 游戏过程中新增计分板组件，动态更新多名玩家的积分和排名状态

### 改进
- **成语详情显示优化** - 成语详情面板中缺失的出处、释义或例子将优雅显示为“无”，而非直接留白
- **界面体验优化** - 修复了消息气泡中文本可能溢出边界的问题，并优化了历史记录详情页的日期与分数排版
- **多人模式得分展示** - 在多人模式下，成语接龙成功后的消息气泡下方也会展示本次接龙的具体得分

## [v1.3.0] - 2026-04-26

### 改进
- **成语数据库更新** - 更新成语库为Li1Fan/chinese-idiom

## [v1.2.1] - 2026-04-18

### 新增功能
- **无效成语交互优化** - 不存在的成语不再显示详情和候选提示，避免误操作

### 改进
- **Debug 版本图标** - Debug 版本角标更新为趣味小黄脸图标，更易区分调试版本

## [v1.2.0] - 2026-04-18

### 新增功能
- **设置页面** - 新增设置页面，支持查看应用信息和检查更新
- **关于页面** - 添加关于页面，展示应用介绍和开源许可证信息

## [v1.1.1] - 2026-04-18

### 新增功能
- **Debug 构建独立标识** - Debug 版本使用独立的 applicationId 和专属图标，可与正式版同时安装

### 改进
- **自动发布 Release** - 推送版本标签时自动创建 GitHub Release 并附带 APK

### 修复
- **限时模式拼音修正** - 修复限时模式下无法修正拼音的问题

## [v1.1.0] - 2026-04-16

### 新增功能
- **数据导入/导出** - 支持导出游戏记录、拼音修正和收藏数据，方便备份和迁移；支持导入已保存的数据文件
- **收藏触觉反馈** - 收藏成语时增加震动反馈，提升操作体验

### 改进
- **版本号显示优化** - 版本字符串格式更新为更标准的 `vX.Y.Z-bBUILD` 格式
